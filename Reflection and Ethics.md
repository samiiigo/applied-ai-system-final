# Reflections and Ethics

My AI system isn't perfect because it is based on data and rules I created, so it can be biased or incomplete. People might misuse this flaw by trying to control the inputs to get a certain result or restrict other options. I would avoid this by making the system more balanced, by testing with different kinds of users and making sure that the inputs are not given too much weight.

One unexpected result from my testing was how the inputs affected the outcome. It didn't take much to cause big shifts. It helped me see the system is reactive but also a bit sensitive.

It was good to work with AI on this project. A good example was when it recommended testing with different profiles, so it was easier to understand the system. At one point though, it provided a code that was supposed to use RAG system but it failed to do so. it was just getting premade data.
